<h2>Frequently Asked Questions</h2>
<h3>How do I change the color scheme?</h3>
<p>You can change the colors of your site really easily using Twenty Fifteen.</p>
<ol>
  <li>In your admin panel, go to Appearance -&gt; Customize.</li>
  <li>Now you will see the Customizer and a tab called &#8216;Colors&#8217;. Click this tab.</li>
  <li>You can now change your color scheme by selecting one of the predefined ones. Choose a color scheme you want from
    Base Color Scheme dropdown. You can preview the change in the Customizer.</li>
  <li>Should you wish to create your own color scheme or modify an existing one, you can by selecting the colors for
    each area listed.</li>
  <li>Once you are happy with your color changes you can click save and your changes will be reflected on your live
    site.</li>
</ol>
<h3>How do I add the social links to the sidebar?</h3>
<p>Twenty Fifteen allows you display links to your social media profiles, like Twitter and Facebook, with icons.</p>
<ol>
  <li>Create a new Custom Menu, and assign it to the Social Links Menu location.</li>
  <li>Add links to each of your social services using the Links panel.</li>
  <li>Icons for your social links will automatically appear if it&#8217;s available.</li>
</ol>

<h3>How do I add a description for my menu link in navigation?</h3>
<p>Twenty Fifteen allows you to not only show a menu link title but also add a description for each menu item.</p>
<ol>
  <li>Visit the Menus page in your admin.</li>
  <li>Use the Screen Options tab to &#8220;Show advanced menu properties&#8221;.</li>
  <li>Select &#8220;Description&#8221; there to start editing menu descriptions.</li>
  <li>Select the menu you want to add links and descriptions to.</li>
  <li>When in the Menu Structure section, you can click open the link and add a description.</li>
  <li>Once you save the menu with your link, the new description should show up.</li>
</ol>
